Bank Simulation Project using Python
This is a Python-based Bank Simulation project that mimics basic banking operations 
such as account creation, deposit, withdrawal, balance inquiry, and transaction history.
The project is designed for learning core Python concepts in a real-world scenario.
Features
- Create new bank accounts
- Deposit and withdraw money
- Check account balance
- View transaction history
- User-friendly console interface
- Data handling using Python logic
 Requirements
- Active internet connection
- Gmail App Password (for email-related features only)
